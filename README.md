# Setup

## Folder structure

```
├── dataset
|  └── trec07p
|     ├── data
|     ├── delay
|     ├── full
|     ├── partial
|     └── README.txt
├── preprocess.py
└── requirements.txt
```

Get the data set from https://plg.uwaterloo.ca/~gvcormac/treccorpus07/ and extract it

## Pre-requisites

Python 3.7.2 (un-tested on other versions)

## Installation

```
pip install -r requirements.txt
```
